About This Application

Overview

This is a simple Flask web application that demonstrates gRPC communication. The application acts as a client that connects to a gRPC server to send greeting requests and display responses.

Technology Stack

  • Flask: Web framework for Python
  • gRPC: High-performance RPC framework
  • Protocol Buffers: Language-neutral serialization mechanism
  • Pipenv: Python dependency management
  • SSL/TLS: Secure communication with certificates

Features

  • Simple greeter service via gRPC
  • Secure communication using SSL certificates
  • Error handling for failed gRPC calls
  • Clean web interface with Flask
  • Automatic gRPC stub generation

Project Structure

The application consists of:

  • app.py - Main Flask application
  • greet.proto - Protocol buffer definition
  • generate_grpc_stubs.sh - Script to generate gRPC stubs
  • templates/ - HTML templates
  • generated/ - Auto-generated gRPC Python code
